Project Goal:

The main goal of the project is to develop a system that measures an individual's heart rate and allows them to monitor the data in real-time. This system can be used as a portable, low-cost health monitoring device. It specifically aims to offer a practical solution for individuals who care about heart health, as well as athletes. The system goes beyond traditional heart rate monitor devices by utilizing IoT (Internet of Things) technologies. To achieve this, heart rate data is sent to the Blynk platform via Wi-Fi, allowing users to easily monitor their data on their smartphones. As a result, users can access heart rate data anytime, anywhere, and analyze historical data.
